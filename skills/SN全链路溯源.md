# SN全链路溯源

## 元信息
- **版本**: 1.0
- **创建时间**: 2026-04-09
- **最后更新**: 2026-04-09
- **触发条件**: 用户提到SN号、序列号、溯源、追溯、SN查询

## 描述
通过SN序列号查询产品的生产、出货、客退全链路质量数据和关键物料信息，定位问题环节。

## 知识
- SN是产品的唯一序列号标识，贯穿生产到售后全流程
- sn_quality_data 表包含该SN的生产/出货/客退质量数据
- sn_quality_key_material 表包含该SN使用的关键物料信息（可用于追溯物料批次问题）
- 如果SN有客退记录，应关注退货原因、复测结果、不良原因等字段
- 如果多个SN使用了同一批次物料且都出现问题，可能是物料批次问题

## 流程
1. 调用 `sn_full_trace` 工具，传入SN号
2. 分析质量数据：是否有客退记录、不良原因是什么
3. 分析关键物料：使用了哪些物料、是否有异常批次
4. 给出综合判断

## 输出格式
### SN基本信息
展示SN的产品名称、生产工厂、生产日期等基本信息。

### 质量数据
展示该SN的客退/不良记录，包括退货原因、复测结果、不良原因、处理状态。

### 关键物料
用表格展示该SN使用的关键物料清单，包括物料名称、供应商、批次等。

### 分析结论
综合以上信息给出该SN的问题定位和建议。
