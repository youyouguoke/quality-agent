# 代工厂质量分析

## 元信息
- **版本**: 1.0
- **创建时间**: 2026-04-09
- **最后更新**: 2026-04-09
- **触发条件**: 用户提到代工厂、工厂质量、生产工厂、工厂对比

## 描述
对代工厂进行质量分析，包括整体质量数据和月度趋势。

## 知识
- 代工厂质量涵盖PQC（制程检验）和OQC（出货检验）等环节
- 关键指标：直通率、不良率、退货率
- 不同工厂生产同一SKU时，可通过对比发现工厂端的质量差异

## 流程
1. 调用 `factory_overview` 工具获取工厂的整体质量数据和月度趋势
2. 分析整体质量指标
3. 分析月度趋势
4. 给出综合评价

## 输出格式
### 工厂基本情况
展示工厂名称及关键质量指标。

### 月度趋势
描述近几月的质量变化趋势。

### 综合评价与建议
给出评价和改善建议。
