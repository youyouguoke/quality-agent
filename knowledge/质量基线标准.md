# 质量基线标准

Agent 分析数据时应参考以下基线标准进行对比判断。当实际指标偏离基线时，应在报告中明确标注异常并给出预警。

## SKU 退货率基线

| SKU类别 | 正常范围 | 预警线 | 严重线 | 备注 |
|---------|---------|--------|--------|------|
| 空气净化器 | ≤2.0% | >2.0% | >4.0% | 含所有型号 |
| 新风机 | ≤1.5% | >1.5% | >3.0% | |
| 加湿器 | ≤2.5% | >2.5% | >5.0% | 季节性波动较大 |
| 扫地机器人 | ≤3.0% | >3.0% | >5.0% | 新品首年可放宽至4% |
| 风扇 | ≤1.0% | >1.0% | >2.0% | |
| 取暖器 | ≤1.5% | >1.5% | >3.0% | |
| 默认 | ≤2.0% | >2.0% | >4.0% | 未明确分类的产品 |

## 供应商 IQC 直通率基线

| 指标 | 优秀 | 合格 | 预警 | 不合格 |
|------|------|------|------|--------|
| 直通率 | ≥99.5% | ≥99% | 98%-99% | <98% |
| 来料退货率 | ≤0.5% | ≤1.0% | 1.0%-2.0% | >2.0% |

## 代工厂质量基线

| 指标 | 优秀 | 合格 | 预警 | 不合格 |
|------|------|------|------|--------|
| 直通率 | ≥98% | ≥95% | 90%-95% | <90% |
| OQC不良率 | ≤0.3% | ≤0.5% | 0.5%-1.0% | >1.0% |

## 客退分析预警规则

| 规则 | 阈值 | 说明 |
|------|------|------|
| 单SKU月度退货环比增长 | >30% | 连续2月增长需重点关注 |
| 单一不良原因占比 | >40% | 存在集中性不良，需专项改善 |
| 供应商直通率 | <99%预警 <98%严重 | 直通率低于98%需立即改善 |
| SKU复测完成率 | <80%预警 <60%严重 | 低于60%需立即催办 |
| 单一供应商责任占比 | >50% | 供应商质量风险集中 |
| 单一物料不良占比 | >40% | 物料批次问题可能性大 |

## 趋势判断标准

| 趋势类型 | 判断规则 |
|----------|---------|
| 改善 | 连续3个月指标向好（退货率下降/合格率上升） |
| 恶化 | 连续2个月指标向差 |
| 波动 | 近3个月内有升有降，幅度超过±10% |
| 稳定 | 近3个月波动幅度在±10%以内 |
