# 质量专业术语

Agent 回答用户问题时应使用准确的质量专业术语，并在必要时向用户解释。

## 检验环节

| 术语 | 全称 | 说明 |
|------|------|------|
| IQC | Incoming Quality Control | 来料检验，对供应商来料进行抽检 |
| PQC | Process Quality Control | 制程检验，生产过程中的质量控制 |
| OQC | Outgoing Quality Control | 出货检验，产品出厂前的最终检验 |
| IPQC | In-Process Quality Control | 制程巡检，在生产现场进行的定期检查 |

## 质量指标

| 术语 | 说明 |
|------|------|
| 直通率 | 产品一次通过全部检验工序的比率，越高越好 |
| 批次合格率 | 检验批次中合格批次的占比 |
| 不良率 | 不合格品数量占检验总数的比率 |
| 退货率 | 客户退货数量占销售总数的比率 |
| PPM | Parts Per Million，百万分之不良率 |
| DPMO | Defects Per Million Opportunities，每百万机会缺陷数 |

## 客退相关

| 术语 | 说明 |
|------|------|
| 复测 | 客退机返回工厂后重新测试，确认故障现象 |
| NTF | No Trouble Found，复测未发现故障 |
| 翻新/返修 | 对确认故障的客退机进行维修恢复 |
| 三包 | 包修、包换、包退的售后服务政策 |
| 7天无理由 | 消费者权益保护法规定的7天无理由退货 |
| DOA | Dead On Arrival，开箱即损 |

## 不良分类

| 分类 | 说明 | 严重等级 |
|------|------|---------|
| 致命不良 (Critical) | 影响安全或导致产品完全无法使用 | 最高 |
| 严重不良 (Major) | 影响主要功能但不涉及安全 | 高 |
| 一般不良 (Minor) | 外观瑕疵等不影响功能的问题 | 低 |

## 责任归属

| 归属方 | 说明 |
|--------|------|
| 来料不良 | 供应商提供的物料存在质量问题 |
| 制程不良 | 代工厂生产过程中引入的缺陷 |
| 设计缺陷 | 产品设计本身存在的问题 |
| 运输损坏 | 物流运输过程中造成的损坏 |
| 用户误操作 | 用户使用不当导致的问题 |
| NTF | 未发现明确故障，可能是误退 |
